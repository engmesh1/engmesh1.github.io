# Meshari Portfolio (Interactive)

An interactive, high-polish portfolio built with **Next.js (App Router)** + **TypeScript** + **Tailwind CSS**.
Includes:
- Dark/Light theme toggle (persisted)
- Animated hero + sections (Framer Motion)
- Project filtering + project details modal
- Keyboard-friendly navigation + accessible components
- SEO + OpenGraph metadata

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Customize
Edit your data in:
- `src/lib/profile.ts`
- `src/lib/projects.ts`
